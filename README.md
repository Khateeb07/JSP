# 📌 About

A hands-on repository containing implementations of core **JSP (JavaServer Pages)** concepts created while learning Java EE web development.

The primary goal of this repository is to understand how JSP works internally and how it integrates with Servlets, JDBC, sessions, and dynamic web applications through practical implementation of individual concepts.

# ▶️ Running the Project

## Requirements

- Java JDK 8+
- NetBeans IDE
- Apache Tomcat or GlassFish Server
- MySQL Server

### Clone the Repository

```bash
git clone https://github.com/khateeb07/JSP.git
```

### Database Configuration

Create a local `db.properties` file and add your database credentials:

```properties
db.url=jdbc:mysql://localhost:3306/your_database
db.username=your_username
db.password=your_password
```

> The `db.properties` file is intentionally excluded from version control.

### Run the Project

- Start Tomcat/GlassFish Server
- Open the project in NetBeans
- Build and Run the application
